# متجر الموضة - Fashion Store 🛒

هذا مشروع موقع تجارة إلكترونية (E-commerce) عصري مخصص للموضة، صاوبتو باستعمال تقنيات حديثة.

## 🚀 التقنيات المستعملة (Technologies):
- **React.js** مع **Vite** (لسرعة الأداء).
- **Tailwind CSS** (للتصميم والديزاين).
- **Vercel** (للنشر والاستضافة).

## ✨ المميزات (Features):
- عرض المنتجات بطريقة عصرية.
- متجاوب مع جميع الشاشات (Mobile & Desktop).
- إضافة المنتجات للسلة (Cart functionality).

## 🛠️ كيفاش تشغل المشروع عندك:
1. `npm install` (باش تنزل المكتبات).
2. `npm run dev` (باش تبدأ الخدمة).
